## 包管理器命令对比

| 操作 | Arch (pacman) | Ubuntu/Debian (apt) | CentOS/RHEL (yum/dnf) |
|------|---------------|---------------------|----------------------|
| 更新系统 | `pacman -Syu` | `apt update && apt upgrade` | `dnf update` |
| 安装软件 | `pacman -S 包名` | `apt install 包名` | `dnf install 包名` |
| 删除软件（保留依赖） | `pacman -R 包名` | `apt remove 包名` | `dnf remove 包名` |
| 删除软件（含依赖） | `pacman -Rs 包名` | `apt autoremove 包名` | `dnf remove 包名`（自动处理依赖） |
| 搜索软件包 | `pacman -Ss 关键词` | `apt search 关键词` | `dnf search 关键词` |
| 查看包信息 | `pacman -Qi 包名` | `apt show 包名` | `dnf info 包名` |
| 查看已安装包 | `pacman -Q` | `apt list --installed` | `dnf list installed` |
| 刷新包数据库 | `pacman -Sy` | `apt update` | `dnf makecache` |
| 清理缓存 | `pacman -Sc` | `apt autoremove` | `dnf autoremove` |
| 清理所有缓存 | `pacman -Scc` | `apt autoclean` | `dnf clean all` |
| 查看包依赖 | `pacman -Si 包名 \| grep Depends` | `apt depends 包名` | `dnf deplist 包名` |
| 查询文件属于哪个包 | `pacman -Qo 文件路径` | `dpkg -S 文件路径` | `dnf provides 文件路径` |
| 仅下载不安装 | `sudo pacman -Sw 包名` | `apt download 包名` | `dnf download 包名` |
| 强制刷新数据库 | `sudo pacman -Syy` | `apt update`（无强制选项） | `dnf makecache` |

## 常用操作等效命令

| 操作 | Arch (pacman) | Ubuntu/Debian (apt) | CentOS/RHEL (dnf) |
|------|---------------|---------------------|-------------------|
| 查看孤儿包 | `pacman -Qtdq` | `apt autoremove`（会直接删除） | `dnf autoremove`（会直接删除） |
| 重新安装 | `sudo pacman -S 包名 --force`（已弃用） | `apt install --reinstall 包名` | `dnf reinstall 包名` |
| 升级指定包 | `sudo pacman -S 包名` | `apt install 包名` | `dnf update 包名` |
| 查看可升级包 | `pacman -Qu` | `apt list --upgradable` | `dnf check-update` |

## 记忆口诀

| 发行版 | 更新系统 | 安装 | 删除 | 搜索 |
|--------|---------|------|------|------|
| **Arch** | `Syu` | `S` | `Rs` | `Ss` |
| **Ubuntu** | `update && upgrade` | `install` | `remove` | `search` |
| **CentOS** | `update` | `install` | `remove` | `search` |

**Arch 的 pacman 三字母记忆法：**
- `-S` = **S**ync（同步/安装）
- `-Q` = **Q**uery（查询本地）
- `-R` = **R**emove（移除）
- `-y` = 刷新数据库
- `-u` = 升级系统
- `-s` = 搜索
- `-i` = 信息
